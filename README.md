# 사용한 기술 및 라이브러리
* Presentation Layer
  - Activity
  - Fragment
  - ViewModel
  - LiveData

* Repository Layer
  - Remote Data Source(Retrofit2)
  - Local Data Source(Room)

* Common
  - Coroutines
  - Android X
  - Koin (for Light-Weight DI)
  - View-Binding

* Third-Party Library
  - Glide, Glide-Transformation
  - Retrofit2
  - OkHttp3
  - Firebase Auth
